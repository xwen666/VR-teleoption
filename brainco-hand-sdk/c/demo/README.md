# C/C++ Demo Examples

Revo1/Revo2 demo programs. New-generation demo programs live in the dedicated SDK.

| Demo | Description |
|---|---|
| `auto_detect` | Auto-detect connected devices |
| `hand_demo` | Basic motor control |
| `hand_monitor` | Real-time monitor |
| `hand_dfu` | Firmware upgrade |
| `debug_detect` | Detection diagnostics |

Build from this directory with `make`, or from `examples/c` with `make`.
