# Python GUI

PySide6 GUI for Revo1/Revo2 devices. New-generation GUI functionality lives in the dedicated SDK.

```bash
cd examples/python/gui
python main.py
python main.py --mock revo2
```

The GUI includes connection, motor control, touch panels, DFU, data collection, timing tests, and system settings for supported Revo1/Revo2 devices.
